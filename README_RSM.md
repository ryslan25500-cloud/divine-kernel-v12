RSM-COIN is a next-generation cryptocurrency integrated with Divine Kernel

### Quick Start
```bash
# Start RSM Blockchain
cd rsm-autonomous && docker-compose up -d

# Run AGI Minter
cd src/rsm-integration && node agi-minter.js
```
